SCUTTLEPy
=========

**SCUTTLEPy** is a Python library for controlling the SCUTTLE robot.

Install
-------

Pip Install
~~~~~~~~~~~

.. code-block:: bash

    pip install scuttlepy


Manual Install
~~~~~~~~~~~~~~

.. code-block:: bash

    git clone https://github.com/ansarid/scuttlepy
    cd scuttlepy
    sudo python3 setup.py install
